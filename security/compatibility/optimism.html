Ledger hardware security devices provide full compatibility with secure layer for secure credential management, authentication, and account protection.

Features

  • Secure secure layer authentication
  • Hardware-isolated credential storage
  • FIDO2/WebAuthn support
  • Offline verification
  • Ledger Security Manager integration
